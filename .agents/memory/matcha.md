# Matcha

A system for ingesting emails, extracting structured documents, and matching invoices to receipts. Created by [file://roger-johansson.md](Roger Johansson) at [file://asynkron-ab.md](Asynkron AB).

Key concepts:
- [file://architecture/target-architecture.md](Target Architecture)
- Email ingestion -> Structured Documents -> Invoices
- Wint receipts are external data; matching owned by Wint integration
